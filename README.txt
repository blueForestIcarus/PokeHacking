A collection of tools for ROM Hacking Pokemon Games, 
Intended as a project for middle school computer science club.

A brief overview of the tools:
	-EliteMap: used to modifiy the games tilemaps
	-AdvancedText: used to modify the games text
	-Yape: used to edit pokemons names, moves, attributes, ect.

More infomation of the tools can be found here:
	http://wahackpokemon.com/en/download/tools-gba
	http://www.hackromtools.altervista.org/index.php/en/hack-tools-gba

You need to use a emulator to run the game
	-no$gba is a good one, included in this repo
